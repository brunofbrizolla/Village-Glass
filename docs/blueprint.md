# **App Name**: VillageGlass Connect

## Core Features:

- Smart Glass Consultation Tool: An AI-powered feature that acts as a consultant to recommend specific glass types (tempered vs. laminated) and thicknesses based on the user's project description and dimensions.
- Dynamic Service Showcase: SEO-optimized sections for Box, Mirrors, Balconies, and Doors featuring lazy-loaded WebP images and semantic H2/H3 hierarchies for maximum search ranking.
- One-Touch WhatsApp Integration: Persistent, floating CTA that links directly to the sales team with a pre-defined inquiry message for high conversion rates.
- Rapid Quote Engine: A streamlined multi-step lead form designed for mobile users to quickly input project details and request estimates.
- Maintenance Service Portal: Specific landing area focused on emergency glass repair and component replacement (pulleys, floor springs) with high-intent keywords.
- Interactive Location Mapper: Integrated Google Maps interface allowing clients to navigate easily to the Curitiba headquarters.
- Lead Management System: A database integration to store and organize budget requests and contact information collected through the site forms.

## Style Guidelines:

- Primary color is a Deep Sophisticated Purple (#6334A6) providing a premium look that contrasts sharply with light text.
- The background uses a darkened desaturated variation (#140E1C) to maintain the brand identity while ensuring low eye strain.
- Accent color is a vibrant Petrol Blue (#3DA9FC) used specifically for CTA buttons and highlighted contact numbers to drive action.
- Headline font is 'Poppins' for a contemporary, precise, and avant-garde look. Body text uses 'Inter' for high readability and a neutral, machined aesthetic as requested for SEO speed.
- Use minimalist, thin-line icons representing glass types, security, and residential environments to reflect professional craftsmanship.
- A Mobile-First approach with high whitespace for text-heavy sections like maintenance and balconies, utilizing CSS Grid for clean card alignment.
- Subtle fade-in effects for service cards and smooth scrolling navigation to improve the user's perceived performance and luxury feel.
